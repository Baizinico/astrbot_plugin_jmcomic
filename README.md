# JMComic下载器

一个用于AstrBot的JMComic漫画下载插件，支持图片和PDF两种发送方式。

## 功能特性

- 🎨 **图片模式**：以合并转发消息形式发送图片，方便浏览
- 📄 **PDF模式**：将所有图片合并为PDF文件发送，PDF密码自动设置为作品ID
- 🚀 **多线程下载**：支持多线程并发下载，提高下载速度
- 🔄 **自动清理**：发送完成后自动清理临时文件

## 安装

### 依赖安装

```bash
pip install -r requirements.txt
```

依赖包：
- `jmcomic` - JMComic下载库
- `Pillow` - 图片处理库
- `pypdf` - PDF处理库

### 插件安装

将插件文件夹放置在 AstrBot 的 `data/plugins/` 目录下即可。

## 使用方法

### 基本命令

```
/jm <作品ID>
```

或

```
/JM <作品ID>
```

### 使用示例

```
/jm 123456
```

插件将自动下载并发送该作品。

## 配置说明

在插件配置文件 `_conf_schema.json` 中可以设置以下选项：

### send_mode（发送方式）

- **image(图片模式)**：以合并转发消息形式发送图片（默认）
- **pdf(PDF模式)**：将所有图片合并为PDF文件发送

## 技术细节

- 下载路径：`/AstrBot/data/Download/<作品ID>/`
- 图片格式：自动转换为PNG格式
- PDF加密：PDF模式下自动使用作品ID作为密码
- 线程配置：图片下载16线程，相册下载8线程

## 注意事项

1. 请确保网络环境可以访问JMComic
2. PDF模式下，文件发送完成后会自动删除临时文件
3. 图片模式下，发送完成后会自动删除下载的图片文件夹
4. 建议在网络状况良好时使用，避免下载超时
